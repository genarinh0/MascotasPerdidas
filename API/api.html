Endpoints

Descripcion

Publicar una Mascota

El usuario debe llenar campos acerca de su mascota:

  • Especie
  • Raza
  • Tamaño
  • Colores
  • Ubicacion Aproximada
  • Fecha Aproximada de Perdida
  • Horario de Contacto
  • Fotografia
  • Descripcion

Ejemplos

POST

/publicar


Request Sample


{

  • "especie": "perro",
  • "raza": "mestizo",
  • "tamaño": "mediano",
  • "colores":["blanco", "cafe"],
  • "ubicacion_aproximada": ["Moderna"],
  • "fecha_aproximada_perdida": "2026/12/08",
  • "horario_contacto": "9:00-21:00",
  • "fotografia": "fotografia.jpg",
  • "descripcion": "Perrito amigable con collar que responde al ser llamado Aldo"

}


Response Sample


200

{

  • "id": 2026,
  • "fecha_creacion": "2026/03/05",

}
